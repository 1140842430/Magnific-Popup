Magnific Popup is a responsive jQuery lightbox plugin that is focused on performance and providing best experience for users with any device and works with jQuery.

  • Read the documentation
  • Configure & download build
  • Contribute to project on Github

What makes it different?

Light and modular

Plugin is modular - you can include in build only features that you neeed using online build tool or by compiling it by yourself with Grunt.js. Size of core JS file: 2.8KB gzipped + each module weights about 0.5KB. Sass CSS preprocessor is used for easier skinning, but you're not obligated to use it.

Content is resized mostly with CSS

Majority of lightbox plugins require you to define size of it via JS option. Magnific Popup does not - feel free to use relative units like EM's or resize lightbox with help of CSS media queries.
Update content inside lightbox without worrying about how it'll resize and center.

Fast

Magnific Popup displays images before they're completely loaded to take full advantage of stuff like progressive JPEG's.
Plugin does not use slow jQuery animations, and provides an easy way to add CSS3 transitions.

Conditional lightbox technique

Plugin has an option to automatically switch to alternative mobile-friendly source on small screen sizes. Brad Frost has a terrific article about this technique.

No external controls

By default controls are made just with help of CSS, without external graphics, which makes them not just light, but also high-dpi screen (retina) ready, but of course you can style them by yourself however you wish.

API + documentation

Tons of options to configure anything, public methods, events, ability to extend any function without modifying the core and other awesome stuff that you like.

Browser support

IE support is under development. Plugins works and tested on modern browsers, iOS, Android, Blackberry. If you found that on some device it works incorrectly, please open issue on GitHub.
